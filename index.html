What is Markdown?

Markdown is a lightweight markup language created by John Gruber in 2004. It allows you to write using an easy-to-read, easy-to-write plain text format, which then converts to valid HTML.

Markdown is widely used for:

  • README files and documentation on GitHub
  • Forum & blog posts
  • Static site generators
  • Note-taking applications

Tables

| Left-aligned | Center-aligned | Right-aligned |
